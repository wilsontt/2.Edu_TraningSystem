## 📋 開發內容

### T4.1: 後端 API - 單筆成績詳情

#### 建立 `/api/exam/record/{record_id}/detail` API

**功能**：
- ✅ 基本資訊：
  - 考生姓名、員工編號、部門
  - 訓練計畫名稱、日期
  - 考試分數、通過狀態
  - 作答時間、提交時間
  - 重考次數
- ✅ 答題詳情：
  - 每題的題目內容
  - 使用者答案 vs 正確答案
  - 是否答對
  - 配分與得分

**權限控制**：
- ✅ 使用者只能查看自己的成績詳情
- ✅ Admin 可查看所有成績詳情

**API 參數**：
- `record_id` (path): 考試記錄 ID

**回應格式**：

```json
    "start_time": "2026-01-09T10:00:00",

  "question_details": [
    {
      "question_id": 1,
      "question_number": 1,
      "content": "題目內容...",
      "question_type": "單選",
      "options": "{\"A\": \"選項A\", \"B\": \"選項B\"}",
      "correct_answer": "A",
      "user_answer": "A",
      "is_correct": true,
      "points": 10,
      "earned_points": 10
    },
    ...
  ]
}
```

## 🔧 技術實作細節

### 後端 API 實作

**檔案**：`backend/app/routers/exam_center.py`

**新增 API 端點**：
- `GET /api/exam/record/{record_id}/detail` - 單筆成績詳情

**資料查詢邏輯**：
1. 查詢 ExamRecord
2. 權限檢查（使用者只能看自己，Admin 可看所有）
3. 查詢 User 和 Department 資訊
4. 查詢 TrainingPlan 資訊
5. 查詢 ExamDetail 和 Question（JOIN）
6. 計算每題得分（答對得滿分，答錯得 0 分）

**題目卡片設計**：
- 答對題目：綠色邊框和背景（`border-green-300 bg-green-50`）

- 使用 `JSON.parse()` 解析選項 JSON 字串
- 顯示選項鍵值對（例如：A: 選項內容）

- 答錯題目在表格中高亮顯示（紅色背景）

- 調整列印時的 padding 和間距

## 🚀 使用方式

### 查看成績詳情

1. 在「個人成績歷史」頁面
2. 點擊任何一筆記錄的「查看詳情」按鈕
3. 開啟成績詳情模態框，顯示：
   - 基本資訊（考生、計畫、分數等）
   - 答題詳情（每題的答案和得分）

### 預覽成績單

1. 在成績詳情模態框中
2. 點擊「預覽成績單」按鈕
3. 開啟成績單預覽視窗
4. 點擊「列印」按鈕可列印成績單
