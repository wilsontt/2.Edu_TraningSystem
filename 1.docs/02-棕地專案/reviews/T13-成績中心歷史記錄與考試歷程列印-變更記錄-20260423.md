# T13 成績中心歷史記錄與考試歷程列印—變更記錄（2026-04-23）

## 1. 目的

將棕地規格 [`02-棕地專案/規格與計畫/T13 增修功能實作PLAN_測試問題.md`](../規格與計畫/T13%20增修功能實作PLAN_測試問題.md) **第 174–248 行**（含 2026/04/20 歷史記錄調整 #3、接續 2026/04/22 列印行為、2026/04/23 版面與檔名修正）之**實作落點**整理為工程記錄，供維護、Code Review 與 UAT 對照；並與程式內 **T13** 註解互相索引。

## 2. 範圍

- **前端**：個人成績歷史 → 考試歷程 Modal（`PlanHistoryModal`）、列印精靈（`ScorePrintFlow`）、單次成績預覽／瀏覽器列印（`ScoreDetailModal`、`ScoreCardPreview`）。
- **後端**：個人歷程 PDF（`POST /exam/personal/print/pdf` → `render_score_print_pdf_to_buffer` 及歷程表繪製輔助函式）。
- **不含**：Admin 報表 `report_print_pdf` 預設 `document_context` 行為變更（維持 `default`）。

## 4. 名詞解釋

| 名詞 | 說明 |
|------|------|
| **詢問1～3** | 考試歷程 Modal 底部「考試歷程成績列印」精靈：訓練計畫、列印方式、列印員工簽名。 |
| **`document_context`** | `personal_exam_history` 時走歷程專用 PDF 分岔（抬頭、不印筆數、表格式歷程等）。 |
| **`printIncludeExamHistory`** | `ScoreCardPreview` 是否把「考試歷程」表併入瀏覽器列印 HTML；單次成績路徑應為 `false`。 |
