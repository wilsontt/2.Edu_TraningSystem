## 4. 名詞解釋

- 最後修改日期：該檔案最後一次被 Git commit 納入版本控制的提交日期。
- 最後修改 Commit：該檔案最後一次變更所屬的 commit SHA。
- 修正內容：根據該 commit 在該檔案上的 diff 所整理出的實際修改點。

### 5.3 各檔案詳細修正內容

#### A. `frontend/src/components/QRCodeLoginPage.tsx`

- 時間格式補上 `hour12: false`，避免顯示 AM/PM，統一為台灣 24 小時制。
- 刪除 `value = value` 的無效賦值，只保留註解，修正 ESLint 與可讀性問題。

#### B. `frontend/src/components/admin/QRCodeManager.tsx`

- `fetchTokens` 改為 `useCallback(async () => ...)`。
- `useEffect` 內改成 `void fetchTokens()`，依賴改為 `[fetchTokens]`，避免 hooks 依賴警告。
- `filter` 回呼參數補明確寫法，整理 ESLint / 格式問題。
- 顯示 QR token 時間時補上 `hour12: false`。

#### C. `frontend/src/components/admin/ReportDashboard.tsx`

- 新增 `ReportDetailScoreRecord`、`ReportExpandDetailPayload` 型別，取代原本鬆散的 `any` 結構。
- 展開部門與計畫明細時，`response.json()` 明確轉成型別資料。
- 查詢參數改集中組成 `query` 字串，整理部門與計畫報表 API URL 的拼接邏輯。
- 明細列表的送出時間改為 `toLocaleString('zh-TW', { hour12: false })`。
- 個人報表導頁由 `tab=history` 改為 `tab=overview`，並補上 `emp_name`、`dept_name`，修正跳轉後資訊不足與頁籤不符問題。

#### D. `frontend/src/components/exam/CheckInButton.tsx`

- 匯入 `isAxiosError`，將 `catch (err: any)` 改為 `catch (err: unknown)`。
- `fetchStatus` 改為 `useCallback`，並讓 `useEffect` 依賴 `[fetchStatus]`。
- 讀取報到狀態與送出報到時，改用 `isAxiosError(err)` 安全取得後端 `detail` 訊息。
- 報到時間顯示補上 `hour12: false`。

#### E. `frontend/src/components/exam/CheckInPage.tsx`

- 新增 `TrainingPlanRow` 型別，`/training/plans` 回傳資料不再使用 `any`。
- `parseInt(planId, 10)` 明確指定 radix。
- 載入報到狀態與送出報到的錯誤處理統一改為 `unknown + isAxiosError`。
- 時間顯示補上 `hour12: false`。

#### F. `frontend/src/components/exam/ExamDashboard.tsx`

- 匯入 `isAxiosError`。
- 從考試中心直接執行報到時，將錯誤處理由 `any` 改為 `unknown`。
- `alert` 顯示內容改為先以 `isAxiosError(err)` 判斷後再取 `response.data.detail`，避免不安全存取。

#### G. `frontend/src/components/personal/PlanHistoryModal.tsx`

- 匯入 `ExamHistoryItem` 型別，歷程列表不再使用 `any`。
- `fetchDetail` 改為 `useCallback`，並在 `useEffect` 中以 `[isOpen, recordId, fetchDetail]` 管理依賴。
- `response.json()` 明確轉為 `ScoreDetail`。
- 歷程提交時間顯示改為 `zh-TW` 24 小時制。
- 明細按鈕改以 `typeof h.id === 'number'` 判斷可否開啟，補強歷程 ID 不存在時的防呆處理。

#### H. `frontend/src/components/personal/ScoreDetailModal.tsx`

- `fetchDetail` 改為 `useCallback`，並修正 `useEffect` 依賴。
- `response.json()` 明確轉為 `ScoreDetail`。
- 成績提交時間顯示改為 `zh-TW` 24 小時制。
- 文案由「重考次數」改為「考試次數」，修正頁面語意。
