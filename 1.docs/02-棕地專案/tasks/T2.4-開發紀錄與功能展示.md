# 教育訓練系統 - 開發紀錄與功能展示

本文件記錄了系統各階段的開發成果、技術細節與 UI 展示。

---

## [Phase 2] 認證與功能權限系統 (RBAC)

### T2.1 & T2.2: 登入、註冊與驗證碼
我們已完成免密碼登入與線上註冊的完整流程。

*   **高級感登入介面**：採用藍色漸層標頭與毛玻璃 (Glassmorphism) 效果，支援「登入/註冊」模式流暢切換。
*   **三欄式註冊**：前端提供工號、姓名、部門三欄位，後端自動關連資料庫並分配預設角色。
*   **安全性**：強制圖形驗證碼 (Captcha) 驗證，防止暴力解析。

### T2.3: JWT 驗證與權限中介軟體
系統安全性已大幅提升，現支援完整的 Token 認證機制。

*   **JWT 機制**：登入成功後簽發 Access Token，前端 Axios 攔截器自動處理 Header 帶入。
*   **會話恢復 (Auto-login)**：重新整理網頁時，系統會自動透過 Token 驗證身份，實現「無感登入」。
*   **後端守護**：實作 `check_permission` 裝飾器，精確攔截無權限的 API 請求。

### T2.4: 動態導覽列實作
實作了智慧型導覽選單，UI 會根據使用者的權限代碼進行動態渲染。

*   **動態選單**：首頁、訓練計畫、考卷工坊等項目，僅在使用者擁有相關權限時顯示。
*   **後台管理過濾**：「系統管理」下拉選單會依據子功能權限（人員、角色、功能清單）動態過濾項目。
*   **視覺優化**：採用更具質感的高對比藍色調設計，強化了 Logo 與選單的互動效果。
