# 線上教育訓練系統 建議調整內容
> 2026/06/27 增加調整內容。

## 1. 系統名稱改為：線上教育訓練系統，所有的文件內容都需更新。

## 2. 教材庫
### 2.1 列表顯示
  - 第一列：`{標題} / {原檔名}`
  - 第二列：標籤 badge（依標籤名 hash 固定配色）
  - 簡述：不在此欄顯示（或 hover 顯示，擇一）
### 2.2 新增上傳（維持現況）
  - 單次 ≤5 檔 / 100MB；一檔一筆教材紀錄
  - 通用教材 plan_id=null；計畫內上傳帶 plan_id
### 2.3 編輯
  - 觸發：操作欄「編輯」（建議 v1 不做點標題，避免與勾選混淆）
  - UI：與「上傳教材」同一卡片（create / edit）
  - 可改：類型、標題、簡述、標籤
  - 可選：替換檔案（單檔、NAS、replace_in_place；綁定計畫須帶 plan_id）
  - 不可改：plan_id、uploaded_by、uploaded_at
  - 綁定計畫教材：可在教材庫編輯中繼資料與換檔
### 2.4 刪除 **暫定計畫，不實作**
  - plan_id != null：教材庫 **禁止刪除**（後端 403 + 前端隱藏/禁用）
  - 解除綁定後（plan_id=null）方可刪除
  - 【待補】解除綁定入口：訓練計畫編輯頁 / 教材庫編輯頁？
  - 【待補】封存計畫教材是否允許刪除


## 3. 員工註冊：
  1. 註冊申請需經同意。
  2. 當有申請註冊，透過`EMAIL通知`。
    - 寄送至 IT_Department。
    - 所以系統要多一個**郵件寄送的管理**畫面。
  3. 管理者可透過該申請連結直接同意。
  4. 續，或登入系統進入**員工註冊申請管理**的頁面審核同意。
