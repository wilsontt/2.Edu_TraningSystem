# 報到管理、訓練計畫、上傳的考卷的考題、教材庫 的擁有權 新需求

## 有關報到的管理
1. 教育訓練報到要分：**上課前的報到**，及**考試時**的報到，因為有時候的訓練（或政策宣導...之類的）不需要考試。
2. 續1, 如果上課前的報到來不及簽到（比如上課遲到，開始上課了，簽到 QRcode 已關掉，或當天請假...），就要等到考試時再來報到。
3. 續2, 反之，已報要的人就不用在該訓練計畫考試時還要再報到一次。
4. 所以，在上課前的報到時間進行簽到，這時候開啟本次訓練的 QRcode 開放簽到。
   1. 上課的人掃 QRcode 後，進行登入，登入成功後，就算報到成功。
   2. 未報到的人，就等本次訓練的考試開始時進行報到。
5. 當要正試開始考試時，可能過了很久（一小時。。。二小時。。。或不止），這時候在`「訓練計畫管理」清單列表的操作欄`增加：`產生 QRcode` 開始進行考試。
6. 續5, 如果已報到者，進入考試中心，該訓練計畫就不用再進行報到。
7. 在系統管理 - QRcode 管理，要增加當按產生 QRcode 時，要先選擇要啟動那一次的訓練計畫，這樣才能知道要記錄那一個訓練計畫的報到。（或是取消這個功能，改在「訓練計畫管理」的操作欄中）

## 訓練計畫、上傳的考卷的考題、教材庫 的擁有權：
1. 為避免誤刪，所以，這幾項都要有「`owner 擁有者`」，只有 Owner 能刪除，其他的只能看或下載，不能刪。
2. 新增訓練計劃時，**「開課單位」**就是「`owner 擁有者`」。
3. 續2, 此時「新增教材套組」時，所上傳的檔案，其 **「開課單位」** 就是「`owner 擁有者`」。
4. 進入考卷工坊，對訓練計畫上傳考卷TXT，考題會進入歷史題庫裡，所以該考卷對應的訓練計畫的 **「開課單位」** 就是「`owner 擁有者`」。
5. 在教材庫裡所新增的教材套組，要能勾選「開課單位」。


## 歷史題庫、教材庫：
1. 在清單列表中增加「開課單位」欄位。
2. 在搜尋功能中增加可搜尋「開課單位」。

## 以下要調整：
3. 在教材庫中，當編輯某一教材套組（或檔案）時，要鎖定，除非取消編輯，否則不能進行其他的操作。
4. 續3, 在檔案檢視頁籤，操作欄中沒有刪除的功能。
5. 續4, 刪除時要提示。。。。萬一錯刪了，有沒有機會救回來？
