# 教材庫：新增／編輯互斥與綁定訓練計畫勾選 UI

**日期**：2026-07-14  
**狀態**：✅ 已實作  

---

## 1. 目的

1. 考卷工坊—教材庫（及訓練計畫教材區）「新增套組」與「編輯套組」不可同時進行。  
2. 綁定訓練計畫改為一列四欄勾選；僅「進行中」「已過期」可選；配色區分。  
3. **不列已封存供新勾**；若套組已綁定封存計畫（作法 A），以唯讀標籤顯示並於儲存時保留。

---

## 2. 行為定案

| 項目 | 定案 |
|------|------|
| 互斥 | 開新增 → 關閉編輯；開編輯 → 關閉新增；對方開啟時按鈕 disabled |
| 可勾選 | `!is_archived` 且依 `end_date` 分進行中／已過期 |
| 排序 | 進行中 → 已過期 →（同組標題） |
| 配色 | 進行中＝綠；已過期＝橘；既有封存綁定＝灰唯讀 |
| API | `GET /training/plans?status=all`（供封存標題對照；勾選區仍排除封存） |

---

## 3. 落點

- `PlanBindingChecklist.tsx`（新建）  
- `MaterialSetUploadPanel.tsx` / `MaterialSetEditPanel.tsx`  
- `TeachingMaterialLibrary.tsx` / `PlanMaterialsSection.tsx`  
- `types/materials.ts`（`PlanOption.end_date`）  
- `api/teachingMaterials.ts`（`status=all`）

---

## 4. 驗收

- [ ] 新增開啟時無法編輯；編輯開啟時無法新增。  
- [ ] 綁定區一列最多四欄；進行中在前、已過期在後，顏色可辨。  
- [ ] 不可勾選已封存；既有封存綁定以灰標顯示，儲存後仍在 `plan_ids`。  
- [ ] 不選任何可勾選計畫＝通用教材（既有封存唯讀仍保留時除外）。
