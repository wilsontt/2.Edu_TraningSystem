# 教材庫／訓練計畫：綁定一欄、取消新增、全選與勾選顯示

**日期**：2026-07-14  
**分支**：`feature/teaching-library-plan-bind-select-ux`  
**狀態**：✅ 已完成（驗收 2026-07-14）  

---

## 1. 目的

對齊先前 4 項 UX：

1. 訓練計畫側欄綁定清單改**一列一欄**（可讀）  
2. 新增套組加**取消**  
3. 教材庫**本頁全選**  
4. 套組勾選後**可看出哪一筆**（修顯示 bug＋高亮）

---

## 2. 行為

| 項目 | 定案 |
|------|------|
| 綁定 `layout` | 側欄／計畫教材＝`stack`（一欄）；教材庫寬欄＝`grid`（至多兩欄） |
| 取消 | `MaterialSetUploadPanel.onClose` → 關閉新增面板 |
| 全選 | 表頭勾選＋「全選本頁」按鈕；再按則取消本頁 |
| 套組勾選 | `toggleSet` 將詳情 `files` 寫回列表；勾選圖示＋標題左邊框高亮 |
