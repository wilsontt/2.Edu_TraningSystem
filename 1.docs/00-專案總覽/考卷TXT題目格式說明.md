## 1. 目的
定義線上考卷系統「上傳 TXT 題目」與「題庫匯入」所使用的純文字檔格式，供題目編製者依循。

## 2. 範圍
適用於 `ExamStudio`（題目工作室）上傳之 `.txt` 檔，解析器為 `backend/app/services/parser.py` 的 `TXTParser`。

## 3. 檔案基本規範
- 編碼：**UTF-8**
- 副檔名：`.txt`
- 每個欄位獨立一行，行首關鍵字大小寫須符合（`Q:`、`A:`、`ANS:`…）。
- 冒號支援半形 `:` 與全形 `：`（選項與 HINT/LEVEL 皆可）。
- 空白行會被忽略；題目以下一個 `Q:` 作為分隔，**不需空行分隔**。
- 同一題若重複出現同欄位（如兩行 HINT），以**最後一行**為準。

## 4. 欄位說明

| 欄位       | 必填       | 說明                                                                     |
| ---------- | ---------- | ------------------------------------------------------------------------ |
| `Q:`       | 是         | 題幹內容                                                                 |
| `A:`～`Z:` | 選擇題必填 | 選項，依字母順序                                                         |
| `ANS:`     | 是         | 正確答案（規則見下）                                                     |
| `SCORE:`   | 是         | 配分，整數；非整數會被視為 0                                             |
| `HINT:`    | 否         | 作答提示                                                                 |
| `LEVEL:`   | 否         | 難易度：`E`/`M`/`H` 或 `Easy 簡單`/`Medium 適中`/`Hard 難`（不分大小寫） |

## 5. 題型判定規則（由解析器自動判斷）
- **是非題**：無 `A:`~`Z:` 選項，且 `ANS` 為 `Y`/`N`/`T`/`F`/`Yes`/`No`（不分大小寫）。系統自動正規化為 `Y`(是)/`N`(否)。
- **多選題**：有選項，且 `ANS` 含逗號（`A,C`）或多個連寫大寫字母（`AC`、`ABCD`）。
- **單選題**：有選項，且 `ANS` 為單一字母（`A`）。

## 6. 格式範例

### 是非題
```
Q: 資訊安全很重要，對嗎？
ANS: Y
SCORE: 10
HINT: 資訊安全是保護資訊資產的重要措施
LEVEL: E
```

### 單選題
```
Q: 哪項不是資安要素？
A: 機密性
B: 完整性
C: 方便性
ANS: C
SCORE: 10
HINT: 資安三要素是 CIA
LEVEL: M
```

### 多選題
```
Q: 資安要素包含？
A: 機密性
B: 完整性
C: 可用性
ANS: ABC
SCORE: 20
HINT: 記住 CIA 三要素
LEVEL: H
```
