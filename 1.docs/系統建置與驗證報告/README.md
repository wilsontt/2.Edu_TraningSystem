## 說明

本目錄依 [驗收報告建立流程](../驗收報告建立流程.md) 設計，用於存放**綠地階段**之驗收報告與階段完成總結。實務上部分階段區塊可能仍為佔位（待專案依流程補登正式報告檔名與連結）。

**棕地（既有功能增強）**之迭代若未依綠地階段目錄產出獨立「T-X 驗收報告」檔，其**任務單、驗收清單與結案註記**集中於：

- [交付實作文件/README.md](../交付實作文件/README.md)

解讀時請區分：**綠地階段表內「（待填寫）」≠ 全專案未完成**；棕地本波（2026-04～05 成績中心部門路徑）以交付實作文件為準據。
