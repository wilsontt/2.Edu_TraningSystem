## 🔧 技術實作細節

### 時間篩選邏輯
- 支援年度、季度、月份三種篩選方式
- 使用 SQLAlchemy 的 `and_` 條件組合
- 正確處理月份和季度的邊界情況

### 成長率計算
- 自動判斷上期時間範圍：
  - 月份篩選：上個月
  - 季度篩選：上個季度
  - 年度篩選：上一年
  - 無篩選：本月 vs 上月
- 計算公式：`((當前平均分數 - 上期平均分數) / 上期平均分數) * 100`

### 成績分布統計
- 分數區間：0-59, 60-69, 70-79, 80-89, 90-100
- 使用 Python 邏輯計算，避免 SQL 複雜查詢

### 個人排名 Top 10
- 計算每個人的平均分數
- 按平均分數降序排序
- 取前 10 名

### 分頁實作
- 使用 SQLAlchemy 的 `offset()` 和 `limit()`
- 計算總頁數：`(total + page_size - 1) // page_size`
