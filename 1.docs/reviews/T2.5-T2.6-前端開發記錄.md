### 更新介面定義
- `DepartmentStat`: 新增 `dept_id`, `rank`, `growth_rate`, `score_distribution`, `top_users`
- `PlanStat`: 新增 `plan_id`, `completion_rate`, `effectiveness_grade`, `popularity_score`

### API 整合
1. **部門統計 API**：
   - 基本：`/api/admin/reports/department`
   - 進階：`/api/admin/reports/department?include_advanced=true&year=2026&month=1`
   - 詳情：`/api/admin/reports/department/{dept_id}/details?page=1&page_size=10`

2. **計畫統計 API**：
   - 基本：`/api/admin/reports/plan`
   - 詳情：`/api/admin/reports/plan/{plan_id}/details?page=1&page_size=10`

### 展開詳情功能
- 點擊展開按鈕時，會動態載入詳情資料
- 使用快取機制，避免重複載入
- 展開時顯示：
  - 部門：成員成績列表、成績分布圖表、個人排名 Top 10
  - 計畫：考生成績列表

### 時間篩選器
- 支援四種模式：全部時間、年度、季度、月份
- 動態顯示對應的輸入欄位
- 篩選參數會自動傳遞到後端 API

### 成長率顯示
- 使用 `TrendingUp` 和 `TrendingDown` 圖標
- 正成長顯示綠色，負成長顯示紅色
- 格式：`±X.X%`

### 成效評級顯示
- 使用彩色徽章顯示 A/B/C/D 等級
- A: 綠色、B: 藍色、C: 黃色、D: 紅色
