### T3.1: 後端 API - 個人成績總覽

#### 建立 `/api/exam/personal/overview` API

**功能**：
- ✅ 已完成考試數
- ✅ 平均分數
- ✅ 通過率
- ✅ 最佳成績
- ✅ 最差成績
- ✅ 總學習時數（累積作答時間）

**權限控制**：
- ✅ 一般使用者只能查看自己的成績
- ✅ Admin 可查看所有使用者成績（需 `emp_id` 參數）

**API 參數**：
- `emp_id` (Optional[str]): 員工編號（僅 Admin 可使用）

**回應格式**：
```json
{
  "emp_id": "E001",
  "completed_count": 10,
  "average_score": 85.5,
  "pass_rate": 90.0,
  "best_score": 100,
  "worst_score": 60,
  "total_study_time": 7200
}
```

### T3.2: 後端 API - 個人成績歷史

#### 建立 `/api/exam/personal/history` API

**功能**：
- ✅ 所有考試記錄列表（時間倒序）
- ✅ 每場考試的詳細資訊：
  - 考試分數
  - 通過狀態
  - 作答時間
  - 提交時間
  - 重考次數
  - 考試計畫名稱
- ✅ 支援分頁與排序

**API 參數**：
- `emp_id` (Optional[str]): 員工編號（僅 Admin 可使用）
- `sort_by` (str, default="time"): 排序欄位（time/score）
- `order` (str, default="desc"): 排序方向（asc/desc）
- `page` (int, default=1): 頁碼
- `page_size` (int, default=20, max=100): 每頁筆數

**回應格式**：
```json
{
  "emp_id": "E001",
  "total": 50,
  "page": 1,
  "page_size": 20,
  "total_pages": 3,
  "records": [
    {
      "record_id": 1,
      "plan_id": 1,
      "plan_title": "資訊安全教育訓練",
      "score": 95,
      "is_passed": true,
      "start_time": "2026-01-09T10:00:00",
      "submit_time": "2026-01-09T10:30:00",
      "duration": 1800,
      "attempts": 1
    },
    ...
  ]
}
```

### T3.3: 後端 API - 個人學習分析

#### 建立 `/api/exam/personal/analysis` API

**功能**：
- ✅ 擅長領域（哪些分類的成績較好，平均分數 ≥ 80）
- ✅ 需要加強的領域（哪些分類的成績較差，平均分數 < 60）

- ✅ 成績趨勢資料（過去 6 個月，按月統計）

**API 參數**：
- `emp_id` (Optional[str]): 員工編號（僅 Admin 可使用）

**回應格式**：
```json

      "category_id": 2,

```

## 🚀 使用方式

### 訪問個人成績頁面

1. **一般使用者**：
   - 點擊導覽列的「成績中心」
   - 自動顯示個人成績頁面

2. **Admin 使用者**：
   - 點擊導覽列的「成績中心」→ 顯示管理報表
   - 或直接訪問 `/reports/personal` → 顯示個人成績

### 功能說明

1. **總覽 Tab**：
   - 查看個人成績總覽（6 個 KPI 卡片）
   - 查看成績分析提示

2. **歷史記錄 Tab**：
   - 查看所有考試記錄
   - 支援排序和分頁
   - 查看成績趨勢圖

3. **學習分析 Tab**：
   - 查看擅長領域和需要加強的領域
   - 查看學習進度
   - 查看各分類成績分析
   - 查看過去 6 個月成績趨勢
