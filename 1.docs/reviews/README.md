# 測試文件與審查報告目錄

本目錄存放測試手冊、修復記錄、開發／變更追溯與重大事件文件。可執行測試程式位於專案根目錄之 **`tests/`**（見 [tests/README.md](../../tests/README.md)）。

---

## 組織規範

| 類型 | 位置 |
|------|------|
| 測試說明、報告、修復記錄、已知問題 | `1.docs/reviews/` |
| 自動化或指令式測試腳本 | `tests/` |
| **棕地迭代任務單與驗收結案**（成績中心等） | [交付實作文件/README.md](../交付實作文件/README.md) |

### 檔名建議

- `test_*.md` — 測試手冊或測試說明  
- `*測試報告*.md` — 測試結果報告  
- `*修復記錄*.md` — 修復問題之記錄
