## 🔧 技術細節

### recharts 導入修正

**修正前**：
```typescript
let LineChart: any, BarChart: any, ...;
try {
  const recharts = require('recharts');
  LineChart = recharts.LineChart;
  ...
} catch (e) {
  console.warn('recharts not installed');
}
```

**修正後**：
```typescript
import {
  LineChart,
  Line,
  BarChart,
  Bar,
  XAxis,
  YAxis,
  CartesianGrid,
  Tooltip,
  Legend,
  ResponsiveContainer
} from 'recharts';
```

### KPI Modal 實作

**功能特色**：
1. **點擊觸發**：點擊任何 KPI 卡片會打開 Modal
2. **詳細資訊**：顯示 KPI 說明、當前數值、相關統計
3. **關閉方式**：
   - 點擊 Modal 外部區域
   - 點擊關閉按鈕（X）
   - 點擊「關閉」按鈕

**Modal 內容**：
- 基本資訊：說明文字、當前數值
- 相關統計（針對特定 KPI）：
  - 總考試場次：本月新增、進行中、即將到期
  - 總應考人次：本月應考、平均分數、及格率
  - 待考試：完成率、進行中的考試、待補考人數
