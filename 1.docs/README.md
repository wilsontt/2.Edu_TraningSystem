本目錄存放所有開發相關的文件、記錄和計劃。

```

├── 角色與權限管理架構說明.md  # 🔐 角色與權限管理架構及設計說明
├── plans/                 # 實作計劃
├── reports/               # 開發報告
├── reviews/               # 測試與修復記錄
├── tasks/                 # 任務記錄
├── 系統建置與驗證報告/       # 驗收報告與階段完成總結
├── 資料庫遷移/              # 資料庫遷移腳本與指南
└── logs/                  # 日誌檔案
```
