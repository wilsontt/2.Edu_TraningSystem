# 修復成績單列印問題

**建立日期**: 2026-01-13  
**狀態**: 待實作

## 問題分析

### 問題1：macOS 下列印成績單時中文字不見了

**現象**：
- 在 macOS 下列印教育訓練成績單時，中文字不顯示
- 在 Windows 下正常顯示

**原因分析**：
- 目前的 `register_chinese_fonts()` 函數嘗試使用系統字體（如 STHeiti、PingFang 等）
- macOS 下的 TTC 字體處理可能失敗，導致回退到 Helvetica（不支援中文）
- 需要改用更可靠的字體載入方式

**解決方案**：
- 從 Google Fonts 下載並使用 Noto Sans CJK（思源黑體）字體
- 將字體檔案包含在專案中，或從網路下載後快取
- 確保字體在所有平台都能正確載入

### 問題2：列印時沒有勾選「包含個人成績歷史」但卻出現了

**現象**：
- 用戶取消勾選「包含個人成績歷史」後列印
- 列印出來的內容仍然包含個人成績歷史

**原因分析**：
- 在 `ScoreCardPreview.tsx` 中，`analysis` 狀態在載入後會被保存
- 即使 `includeHistory` 設為 false，`analysis` 狀態仍然存在
- `ScoreCardContent` 組件檢查 `includeHistory && analysis`，如果 `analysis` 存在就會顯示

**解決方案**：
- 當 `includeHistory` 為 false 時，不傳遞 `analysis` 給 `ScoreCardContent`
- 或者在 `includeHistory` 變為 false 時清除 `analysis` 狀態
- 確保列印時只使用當前的 `includeHistory` 狀態

## 修復計劃

### 修復1：使用 Google Fonts Noto Sans CJK 字體

**檔案**：`backend/app/routers/report.py`

**修改內容**：

1. **下載 Noto Sans CJK 字體**：
   - 從 Google Fonts 下載 Noto Sans CJK Regular TTF 字體
   - 將字體檔案放在 `backend/fonts/` 目錄下
   - 或使用 `requests` 下載並快取字體檔案

2. **修改 `register_chinese_fonts()` 函數**：
   - 優先嘗試載入專案中的 Noto Sans CJK 字體
   - 如果不存在，從 Google Fonts 下載
   - 確保在所有平台（macOS、Windows、Linux）都能正確載入

**實作方式**：

```python
import requests
import os

def download_noto_font():
    """從 Google Fonts 下載 Noto Sans CJK 字體"""
    font_dir = os.path.join(os.path.dirname(__file__), '..', 'fonts')
    os.makedirs(font_dir, exist_ok=True)
    font_path = os.path.join(font_dir, 'NotoSansCJK-Regular.ttf')
    
    if os.path.exists(font_path):
        return font_path
    
    # 從 Google Fonts 下載
    url = "https://github.com/google/fonts/raw/main/ofl/notosanscjktc/NotoSansCJKtc-Regular.otf"
    # 或使用 TTF 版本
    # url = "https://github.com/google/fonts/raw/main/ofl/notosanscjktc/NotoSansCJKtc-Regular.ttf"
    
    try:
        response = requests.get(url, timeout=10)
        if response.status_code == 200:
            with open(font_path, 'wb') as f:
                f.write(response.content)
            return font_path
    except Exception as e:
        print(f"Failed to download Noto Sans CJK font: {e}")
    
    return None

def register_chinese_fonts():
    """註冊中文字體（優先使用 Noto Sans CJK）"""
    global _registered_chinese_font
    
    if _registered_chinese_font:
        return _registered_chinese_font
    
    font_name = "NotoSansCJK"
    
    # 優先嘗試使用 Noto Sans CJK
    noto_font_path = download_noto_font()
    if noto_font_path and os.path.exists(noto_font_path):
        try:
            pdfmetrics.registerFont(TTFont(font_name, noto_font_path))
            _registered_chinese_font = font_name
            print(f"Successfully registered Noto Sans CJK font: {noto_font_path}")
            return font_name
        except Exception as e:
            print(f"Failed to register Noto Sans CJK font: {e}")
    
    # 如果 Noto Sans CJK 載入失敗，回退到原有的系統字體邏輯
    # ... 原有的系統字體載入代碼 ...
```

**注意事項**：
- 需要確保 `requests` 套件已安裝（通常已包含在 requirements.txt）
- 字體檔案大小約 15-20MB，首次下載可能需要一些時間
- 可以考慮將字體檔案加入版本控制，避免每次下載
