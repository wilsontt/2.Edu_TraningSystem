## 一、目的與適用對象

本文件說明「教育訓練教材及線上考卷系統」的**使用情境、操作步驟與注意事項**，供下列對象使用：

- **系統管理員**：單位、分類、人員、角色、權限、功能清單設定
- **訓練／計畫承辦**：訓練計畫建立與維護、考卷出題、題庫匯入、報到與 QRcode
- **一般使用者**：登入、報到、應考、查詢個人成績與學習分析

詳細架構與技術說明請見 [專案系統架構分析](專案系統架構分析.md)；快速啟動與目錄說明請見 [README](README.md)。

## 二、系統入口與登入

### 2.1 存取網址

- **本機開發**：`http://localhost:5173`（或 Vite 顯示之位址）
- **區網**：`http://<伺服器IP>:5173`（依實際部署設定）

### 2.2 登入方式

| 方式 | 說明 |
|------|------|
| **一般登入** | 輸入員工編號 → 取得圖形驗證碼 → 輸入驗證碼 → 登入 |
| **QRcode 登入** | 由管理員產生登入 QRcode，掃描後進入登入頁，仍須輸入員工編號與驗證碼 |

**注意**：  
- 員工編號需已於「人員管理」中建立；若為首次使用，可能需透過「註冊」或由管理員先行建檔。  
- 驗證碼為 4 位數字，逾時或錯誤需重新取得。

### 2.3 註冊（若開放）

於登入頁選擇「註冊」→ 選擇部門 → 輸入員工編號與姓名等必填欄位 → 送出。註冊後通常需由管理員指派角色與單位，方能使用完整功能。

## 三、管理後台使用情境

### 3.1 系統管理

| 功能 | 說明 | 常見操作 |
|------|------|----------|
| **單位管理** | 部門/單位的新增、修改、刪除、成員檢視與調整 | 新增單位、將使用者指派至單位、查看成員 |
| **分類管理** | 訓練課程「大項目／細項目」階層 | 新增大項目、在項目下新增細項目、供訓練計畫選用 |
| **人員管理** | 使用者之單位、角色、狀態 | 查詢、編輯單位/角色、停用/啟用帳號、分頁與排序 |
| **角色管理** | 角色名稱、成員、權限 | 新增角色、為角色指派功能權限、檢視成員 |
| **權限管理** | 角色可存取的「功能清單」 | 依角色勾選/取消功能，控制選單與按鈕顯示 |
| **功能清單** | 系統功能樹（選單與按鈕） | 維護功能代碼、名稱、父子關係、路由路徑（通常由開發/管理員操作） |

**權限**：以上功能依角色權限顯示；未具權限之選單或按鈕不會出現。

### 3.2 訓練計畫管理

| 功能 | 說明 | 常見操作 |
|------|------|----------|
| **列表與頁籤** | 「正在進行中」「已過期」「已封存」三個頁籤 | 切換頁籤、依年份/開課單位/分類篩選、關鍵字搜尋 |
| **新增／編輯計畫** | 細項目、開課單位、訓練日期、結束日期、及格分數、計時與時間限制、應到人數 | 選擇受課單位（部門）、選擇個人授課對象、儲存 |
| **封存／取消封存** | 將計畫標記為已封存或恢復 | 在列表中依計畫操作「…」→「封存計畫」或「取消封存」，需確認 |
| **刪除計畫** | 刪除訓練計畫（慎用） | 「…」→「刪除計畫」→ 確認 |
| **報到統計** | 應到/實到、報到名單、未報到名單 | 點選「報到統計」→ 檢視或產生報到 QRcode |
| **報到 QRcode** | 供學員掃描報到之 QRcode 與網址 | 在報到統計區產生、複製連結或展示供掃描 |

**注意**：  
- 封存後，該計畫不會出現在「正在進行中」「已過期」，且考試中心、考卷工坊等僅會顯示未封存計畫。  
- 個人授課對象可在編輯計畫時以「員工編號、姓名、部門」搜尋並多選。

### 3.3 考卷工坊

| 功能 | 說明 | 常見操作 |
|------|------|----------|
| **選擇訓練計畫** | 先選定一筆訓練計畫，再管理其題目與教材 | 由下拉或列表選擇計畫（僅顯示未封存） |
| **上傳 TXT 題目** | 依固定格式上傳文字檔，系統解析為單選/多選/是非 | 上傳符合格式之 .txt、確認解析結果、必要時在題目列表中修正 |
| **編輯／刪除題目** | 修改題目內容、選項、答案、配分、提示 | 在題目列表點選編輯或刪除、存檔 |
| **從題庫匯入** | 自題庫挑題加入當前計畫 | 開啟「從題庫匯入」→ 用關鍵字/題型/標籤篩選 → 勾選題目 → 確認匯入 |
| **教材上傳與預覽** | 上傳教材檔案、連結至計畫、預覽 | 上傳符合格式與大小之檔案、在介面中預覽或下載 |

**題目格式概要**：  
- 單選：`Q:` 題目、`A:`/`B:`… 選項、`ANS: A`、`SCORE: 10`  
- 多選：`ANS: A,C` 或 `ANS: AC`  
- 是非：`ANS: Y` 或 `ANS: N`  

詳細格式與範例請參考後端或專案內題目解析說明。

### 3.4 題庫維護

| 功能 | 說明 | 常見操作 |
|------|------|----------|
| **題庫列表** | 查詢題庫內所有題目 | 依關鍵字、題型、標籤篩選、分頁瀏覽 |
| **編輯／刪除** | 修改題庫題目內容或刪除 | 僅影響題庫本身，不直接刪除已匯入考卷的題目 |
| **題型與標籤** | 題型為單選/多選/是非；標籤供分類與篩選 | 篩選時可選題型、輸入標籤關鍵字 |

題庫與考卷題目為不同來源：考卷題目綁定計畫，題庫為共用池；自題庫匯入會將題目複製到該計畫的考卷中。

### 3.5 成績中心（報表）

| 功能 | 說明 | 常見操作 |
|------|------|----------|
| **總覽** | 總考試場次、應考人次、平均分數、及格率等 | 查看數字與圖表 |
| **部門統計** | 各部門應考人次、平均分、及格率、完成率 | 依部門篩選、下鑽至部門成員明細 |
| **計畫統計** | 各訓練計畫之應考與成績摘要 | 依計畫篩選、下鑽至該計畫考生明細 |
| **個人成績查詢** | 查詢指定員工之成績（需權限） | 輸入員工編號或從報表連結進入個人成績 |
| **PDF 導出** | 產出成績相關 PDF | 依畫面上之導出或列印功能操作 |

**權限**：成績中心相關選單與 API 皆受「報表／成績」權限控制。

## 四、考試中心（應考者）

### 4.1 考試列表

登入後進入「考試中心」或「我的考試」：  
- 列出**尚未開始、進行中、已完成、已過期**之訓練計畫（依系統設計分類）。  
- 僅顯示**未封存**且使用者有權應考之計畫（依受課單位/個人判斷）。

### 4.2 報到（若該計畫要求報到）

- 在開始考試前，若有「報到」按鈕或連結，需先完成報到。  
- 可掃描訓練計畫產生之**報到 QRcode** 或點選報到連結，輸入員工編號與驗證碼後完成報到。  
- 報到後再進入「開始考試」。

### 4.3 作答與提交

1. 選擇一筆計畫 → 點「開始考試」。  
2. 依題目順序作答（單選/多選/是非），可依介面設計前後翻頁或跳題。  
3. 若有計時，請在時間內完成。  
4. 作答中離開再回來，若系統有「暫存」設計，會還原至上次進度（依實際實作為準）。  
5. 確認無誤後點「提交」→ 系統即時評分 → 顯示成績與通過與否。  
6. 若未通過且系統允許多次作答，可再選同一計畫重考；次數與規則依系統設定。

## 五、個人成績與學習分析

### 5.1 個人成績總覽

- 路徑通常為「成績中心」→「個人」或側欄「個人成績」。  
- 可查看：已完成考試數、平均分、通過率、最佳/最差成績、總學習時數（若有）等。

### 5.2 歷史紀錄與詳情

- **歷史紀錄**：依計畫或時間列出歷次考試與分數。  
- **成績詳情**：點選單筆紀錄可查看該次考試每題對錯、正確答案與使用者答案。  
- **成績單預覽／列印**：若有「成績單」或「列印」功能，可預覽或列印該次成績單（含 PDF 下載）。

### 5.3 學習分析（若有提供）

- 可能包含：各分類成績分布、歷月表現、待加強項目等。  
- 依畫面提供之篩選與圖表解讀即可。

## 六、常見問題與注意事項

### 6.1 登入與權限

| 問題 | 建議處理 |
|------|----------|
| 驗證碼一直錯誤 | 確認輸入與圖片一致、英文/數字勿混淆；必要時重新整理取得新驗證碼 |
| 登入後沒有選單或功能很少 | 可能角色權限不足，請管理員在「權限管理」中為該角色勾選所需功能 |
| 忘記員工編號 | 由管理員在「人員管理」中查詢或重設（若支援） |

### 6.2 訓練計畫與考卷

| 問題 | 建議處理 |
|------|----------|
| 找不到剛建的訓練計畫 | 確認頁籤（進行中/已過期/已封存）與篩選條件；確認未遭封存 |
| 題目上傳失敗或解析錯誤 | 檢查 TXT 格式是否符合（Q:/A:/B:/ANS:/SCORE: 等）；編碼建議 UTF-8 |
| 從題庫匯入找不到題目 | 用關鍵字、題型、標籤縮小範圍；確認題庫中已有該題 |

### 6.3 考試與成績

| 問題 | 建議處理 |
|------|----------|
| 考試中心沒有可考的計畫 | 確認登入身分是否在被指派之受課單位或個人名單中；確認計畫未封存、且處於「進行中」日期區間 |
| 提交後看不到成績 | 重新進入「個人成績」或該次考試詳情；若仍無，可能為網路或權限問題，需由管理員/維運檢視紀錄 |
| PDF 或列印異常 | 確認瀏覽器允許彈跳視窗與下載；改用 Chrome/Edge 等常見瀏覽器再試 |

### 6.4 資料與系統

- **資料備份**：重要資料庫檔案（如 `education_training.db`）應由管理員定期備份。  
- **瀏覽器**：建議使用 Chrome、Edge、Safari、Firefox 等最新版本，並允許 JavaScript 與 Cookie。  
- **螢幕與裝置**：支援手機、平板、桌機；若為行動裝置，請在穩定網路下操作以利提交與載入。
