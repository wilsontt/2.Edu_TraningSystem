### 問題 2: Department Comparison API ✅ 已修復（2026-01-10）

**原問題**：`/api/admin/reports/department-comparison` - SQLAlchemy 2.0 的 `case()` 語法錯誤

**原錯誤訊息**：
```
The "whens" argument to case(), when referring to a sequence of items, 
is now passed as a series of positional elements, rather than as a list.
```

**修復內容**：
- ✅ 改用 Python 邏輯計算及格數，避免在 SQL 查詢中使用 `case()`
- ✅ 先查詢所有記錄，然後在 Python 中計算及格數
- ✅ 添加註解說明修復原因

**修復檔案**：`backend/app/routers/report.py` (第 570-631 行)

**狀態**：✅ **已修復並驗證通過**
