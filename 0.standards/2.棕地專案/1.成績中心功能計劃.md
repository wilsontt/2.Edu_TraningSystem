## 一、使用者角色與需求分析 (User Roles & Requirements)

### 1.1 管理員（Admin）

**核心需求**：
- 查看全公司整體統計數據
- 分析各部門/各訓練計畫的成效
- 產出管理層級報表
- 追蹤未完成考試的人員
- 識別需要加強的訓練領域

**使用場景**：
- 定期檢視公司整體訓練成效
- 向高階主管報告訓練成果
- 規劃下一階段訓練計畫
- 發現訓練盲點並改善

### 1.2 部門主管（Department Manager）

**核心需求**：
- 查看所屬部門成員的成績
- 了解部門整體表現
- 識別需要補強的人員
- 追蹤部門訓練完成率

**使用場景**：
- 檢視部門成員學習狀況
- 向管理層報告部門訓練成效
- 安排部門內補強訓練
- 評估成員能力發展

### 1.3 一般使用者（User）

**核心需求**：
- 查看個人歷史成績
- 了解個人學習進度
- 下載個人成績單
- 查看個人學習軌跡

**使用場景**：
- 檢視個人考試成績
- 了解個人學習強弱項
- 列印成績單供存檔
- 追蹤個人成長歷程

## 二、功能模組規劃 (Functional Modules)

### 模組 1：總覽儀表板（Dashboard Overview）

#### 1.1 關鍵指標卡片（KPI Cards）

**現有指標**：
- ✅ 總考試場次：所有已完成的訓練計畫數
- ✅ 總應考人次：累積考試人次（含重考）
- ✅ 平均分數：全公司平均分數
- ✅ 及格率：整體及格百分比

**建議新增指標**：
- 📊 **本月新增考試場次**：當月新增的訓練計畫數
- 📊 **本月應考人次**：當月的考試人次
- 📊 **待考試人數**：已指派但尚未完成考試的人員數
- 📊 **平均作答時間**：所有考試的平均作答時間
- 📊 **完成率**：已完成考試人數 / 應考人數
- 📊 **補考率**：需要補考的人次占比

**視覺設計**：
- 使用卡片式設計，每個指標一個卡片
- 使用圖示與顏色區分不同指標類型
- 支援數值變化趨勢顯示（↑↓）
- 點擊卡片可鑽取詳細資料

#### 1.2 趨勢圖表（Trend Charts）

**時間趨勢分析**：
- **過去 6 個月/12 個月的成績趨勢**：折線圖顯示平均分數變化
- **及格率變化**：折線圖顯示及格率隨時間變化
- **應考人次趨勢**：長條圖顯示每月應考人次

**部門對比分析**：
- **各部門平均分數對比**：長條圖或雷達圖
- **各部門及格率對比**：長條圖
- **各部門完成率對比**：長條圖

**計畫熱度分析**：
- **各計畫應考人次排行**：圓餅圖或長條圖
- **各計畫平均分數排行**：長條圖
- **各計畫難度分析**：散點圖（應考人次 vs 平均分數）

#### 1.3 即時狀態（Real-time Status）

**進行中的考試**：
- 目前可進行的考試數量
- 進行中考試列表（含到期時間）

**即將到期提醒**：
- 3 天內到期的考試提醒
- 即將到期考試列表

**待補考名單**：
- 未通過需補考的人員清單
- 補考提醒通知

### 模組 2：多維度統計分析（Multi-dimensional Statistics）

#### 2.1 部門統計（Department Statistics）

**現有功能**：
- ✅ 應考人次
- ✅ 平均分數
- ✅ 及格率

**建議強化功能**：

**進階分析**：
- 📊 **部門內成績分布**：直方圖顯示分數分布情況
- 📊 **部門排名**：與其他部門的對比排名
- 📊 **部門內個人排名**：Top 10 高分人員
- 📊 **部門完成率**：已完成考試人數 / 應考人數
- 📊 **部門成長率**：與上期相比的成長率

**時間篩選**：
- 可選擇特定時間區間（年度/季度/月份/自訂）
- 支援多時間區間對比

**部門詳情**：
- 點擊部門可查看該部門所有成員的詳細成績
- 支援部門成員成績列表與排序

#### 2.2 訓練計畫統計（Training Plan Statistics）

**現有功能**：
- ✅ 應考人次
- ✅ 平均分數
- ✅ 及格率

**建議強化功能**：

**進階分析**：
- 📊 **計畫難度分析**：平均分數分布直方圖
- 📊 **題目答對率分析**：哪些題目錯誤率最高（需後端支援）
- 📊 **計畫完成率**：已考人數 / 應考人數
- 📊 **計畫成效評級**：A/B/C/D 等級評定
- 📊 **計畫熱度**：應考人次與平均分數的綜合評分

**時間篩選**：
- 可選擇年度、季度、月份
- 支援多時間區間對比

**計畫詳情**：
- 點擊計畫可查看該計畫所有考生的詳細成績
- 支援考生成績列表與排序
- 支援題目錯誤率分析

#### 2.3 個人成績查詢（Personal Score Query）⭐ 新增功能

**個人成績總覽**：
- 📊 **已完成考試數**：個人完成的所有考試數量
- 📊 **平均分數**：個人所有考試的平均分數
- 📊 **通過率**：個人通過考試的百分比
- 📊 **最佳成績**：個人最高分數記錄
- 📊 **最差成績**：個人最低分數記錄（需補強領域）
- 📊 **總學習時數**：累積作答時間

**個人成績歷史**：
- 📋 **所有考試記錄列表**：時間倒序排列
- 📋 **每場考試的詳細資訊**：
  - 考試分數
  - 通過狀態
  - 作答時間
  - 提交時間
  - 重考次數
  - 考試計畫名稱
- 📊 **成績趨勢圖**：個人分數隨時間變化折線圖

**個人學習分析**：
- 📊 **擅長領域**：哪些分類的成績較好（依訓練計畫分類）
- 📊 **需要加強的領域**：哪些分類的成績較差
- 📊 **學習進度**：已完成 / 總計畫數
- 📊 **學習軌跡**：個人學習歷程時間軸

#### 2.4 時間維度分析（Time Dimension Analysis）⭐ 新增功能

**年度統計**：
- 各年度的整體表現對比
- 年度成長率分析

**季度統計**：
- 各季度的表現對比
- 季度趨勢分析

**月度統計**：
- 各月份的表現趨勢
- 月度波動分析

**自訂時間區間**：
- 可選擇任意時間範圍進行分析
- 支援多時間區間對比

### 模組 3：詳細成績查詢（Detailed Score Query）

#### 3.1 成績列表查詢

**多條件篩選**：
- 🔍 **訓練計畫**：下拉選單選擇特定計畫
- 🔍 **部門**：下拉選單選擇特定部門
- 🔍 **時間區間**：日期選擇器
- 🔍 **通過狀態**：單選（全部/通過/未通過）
- 🔍 **分數區間**：滑桿或輸入框（最低分 - 最高分）
- 🔍 **關鍵字搜尋**：姓名、員工編號

**排序功能**：
- 依分數排序（高到低/低到高）
- 依時間排序（新到舊/舊到新）
- 依部門排序
- 依計畫排序

**分頁顯示**：
- 支援大量資料分頁
- 可調整每頁顯示筆數（10/20/50/100）
- 顯示總筆數與當前頁數

#### 3.2 單筆成績詳情

**基本資訊**：
- 👤 考生姓名、員工編號、部門
- 📚 訓練計畫名稱、日期
- 📊 考試分數、通過狀態
- ⏱️ 作答時間、提交時間
- 🔄 重考次數

**答題詳情**：
- 📝 每題的題目內容
- ✅ 使用者答案 vs 正確答案
- ✓ 是否答對
- 📊 配分與得分
- ❌ 答錯題目的錯誤分析

**成績單預覽**：
- 📄 可預覽 PDF 格式的成績單
- 🖨️ 支援列印功能

- 分類占比

- 分數分布
- 作答時間分布

**散點圖（Scatter Plot）**：
- 分數與時間的關聯

**維度切換**：
- 圖表可切換不同維度（時間/部門/計畫）

**圖表匯出**：
- 支援圖表匯出為 PNG
- 支援圖表匯出為 PDF
- 支援圖表匯出為 SVG
