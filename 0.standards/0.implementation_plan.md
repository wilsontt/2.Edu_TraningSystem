## 專案範疇 (Project Scope) - MVP

本系統為獨立運行的 Web 應用程式，專為公司內部教育訓練設計。

### 核心技術棧 (Confirmed)
- **前端**: React + TypeScript + Vite (支援行動端響應式介面)
- **後端**: FastAPI (Python) - 輕量、高效，適合本地端運行
- **資料庫**: SQLite (本地檔案儲存)
- **輸出**: PDF 格式 (單排/成績單)
- **部署**: 本地伺服器環境 (Standalone)

## Phase 1: MVP 功能設計

### 1. 使用者與登入 (User Management)
- **註冊機制**: 
    - 第一次使用可線上註冊 (工號/姓名/部門)。
    - **免密碼設計**: 僅需輸入工號。
- **認證機制**: 
    - 登入時輸入「工號 + 圖形驗證碼 (Captcha)」。
    - 系統驗證工號是否存在。

### 2. 訓練與考卷管理 (Management)
- **基礎訓練管理**:
    - 建立訓練計畫名稱 (例如：2025資安教育訓練)。
    - 關連考卷與教材資訊。
- **考卷匯入**:
    - 僅接受 **TXT 檔案** 上傳。
    - 支援線上編輯/新增題目。
- **題庫增加計畫**:
    - [建議] 提供 AI 工具分析 PDF 教材內容，自動產生試題草稿。

### 3. 線上考試與評分 (Exam & Grading)
- **行動端優化**: 介面必須適合手機/平板作答。
- **自動評分**: 交卷後即時從資料庫判斷對錯並計算得分。
- **成績保存**: 儲存至 SQLite，包含作答時間與細節。

### 4. 查詢與輸出 (Reporting)
- **視覺呈現**: 成績顯示採用「紅字手寫/批改」風格 (CSS 特效)。
- **PDF 輸出**: 
    - 依樣張格式。
    - 考試名稱併入日期列最左側。
    - Score 欄位內含及格標註。

## AI 出題實作計畫 (Smart Quiz Generation)
- **後端整合**: 整合 Python `pypdf` 或 `python-pptx` 讀取內容。
- **AI 處理**: 串接 LLM (Gemini/GPT) 生成 JSON 格式題目。
- **前端預覽**: 建立一個「試卷草稿編輯器」，讓管理員在正式儲存前可以微調。
